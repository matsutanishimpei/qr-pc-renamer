# PC名変更ツール（PC Renamer）

## 概要

このツールは、QRコードや手動入力を使ってWindowsのPC（コンピュータ）名を簡単に変更するためのツールです。
**【重要】** 必ず「管理者権限のあるユーザー」でWindowsにログインした状態で実行してください。
（実行時に自動的に権限昇格の確認が出ますが、管理者権限がないユーザーだとパスワードを求められるか失敗する場合があります）

## 使い方

1. `run_renamer_qr.bat` をダブルクリックして実行します。
2. 「このアプリがデバイスに変更を加えることを許可しますか？」という画面（ユーザー アカウント制御）が出たら、「はい」をクリックします。
3. 黒いコマンドプロンプト画面と一緒に、QRコード読み取り用のカメラウィンドウが起動します。

### QRコードを使う場合

- カメラにPC名が書かれたQRコードをかざしてください。
- 読み取りに成功すると画面にPC名が表示され、変更してよいか確認されます。

### 手入力する場合

- `ESCキー` を押してカメラウィンドウを閉じると、コマンドプロンプト画面で手入力モードに切り替わります。
- 「新しいPC名を入力してください」と表示されるので、キーボードから直接入力してEnterを押してください。

### 完了後

1. 変更が完了すると、「✔ コンピュータ名の変更に成功しました」と表示されます。
2. 何かキーを押して画面を閉じます。
3. **※重要※ 変更したPC名を完全に反映させるために、PCを再起動してください。**

## その他・注意点

- PC名を変更した履歴は、「rename.log」というファイルとして自動で保存・追記されていきます。
- 実行時に「現在の PC 名」と「新しい PC 名」が同じ場合は、変更処理を行わずに終了します。
